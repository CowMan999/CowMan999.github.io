About Me

Hardware head,
software soul.

I'm Edward Mayen, a Computer Engineering student at McMaster University, based in Mississauga, Ontario. I'm passionate about systems that bridge the physical and digital worlds — from bare-metal firmware on microcontrollers to full IoT deployments and custom PCB hardware.

My focus areas include embedded systems, IoT architecture, systems programming, and PCB design. I enjoy working close to the metal — understanding how hardware behaves lets me write software that's leaner, faster, and more reliable.

Whether it's designing a sensor node from schematic to soldered board, writing a real-time driver, or connecting edge devices to cloud infrastructure, I care about building things that actually work in the real world.

"Good embedded engineering means understanding both the datasheet and the deadline."

RISC-V Processor on FPGA ✦ Featured
Designed and implemented a pipelined 32-bit RISC-V processor in VHDL, complete with hazard detection, forwarding unit, and a custom branch predictor achieving 94% accuracy. Synthesized on a Xilinx Artix-7 FPGA with a 5-stage pipeline running at 100 MHz. Includes a custom assembler written in Python and a UART-based debug interface.
